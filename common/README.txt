This directory contains the implementation of some modules shared by the solvers.

Coded by Zaikun ZHANG (www.zhangzk.net) based on Powell's code and papers.

Due to the dependence among the modules, the files must be compiled in the order shown in ffiles.txt.

Dedicated to the late Professor M. J. D. Powell FRS (1936--2015).
